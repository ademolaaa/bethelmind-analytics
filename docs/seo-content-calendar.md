Internal Linking Rules
- Every blog post links to 1–2 solution pages + one conversion page (/whatsapp or /booking).
- Every solution page links to 1–2 related blog posts (implementation guide + templates/checklist).
- Use descriptive anchors:
  - Good: “WhatsApp CRM & Sales Automation” / “invoice payment reminders”
  - Avoid: “click here”, “learn more”

On-page Standards
- One H1 per page (title)
- Use H2 for main sections; H3 for sub-sections
- Include FAQ section when there are repeated objections; emit FAQPage schema
- Add CTAs near the top and near the bottom of content

Performance Targets
